# Sistema de Gestion de Barrios Privados

Base inicial para la Fase 2 del TPO.

## Descripcion

El proyecto prepara una estructura para modelar un sistema de gestion de barrios privados.

## Tecnologia

- Java 21 LTS
- Maven

## Estructura de paquetes

com.barrios.modelo
com.barrios.servicio
com.barrios.notificacion
com.barrios.estado
com.barrios.factory
com.barrios.proxy
com.barrios.main


## Como compilar

mvn clean compile


## Como ejecutar

mvn exec:java
